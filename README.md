# 🌐 AI Web Summarizer

Hey! 👋
This is a small project I built while learning web development and Python — a simple app that takes a webpage link and gives you a quick summary.

I wanted to understand how real websites work behind the scenes, so I tried building one myself… and this is the result 😄

💡 What this project does
You just:
1. Paste any webpage URL
2. Click **Summarize**
3. Get a short version of the content in seconds

⚙️ How it actually works
* It fetches the webpage using Python
* Cleans all the messy HTML stuff
* Extracts useful text (mainly paragraphs)
* Picks the most meaningful sentences
* Shows them neatly on a webpage

Not super “AI” yet 😅 — but a solid step towards it!

🛠️ What I used
* Python
* Flask (to build the website)
* BeautifulSoup (to read webpages)
* Requests
* HTML / CSS
* Render (for deployment)
* Gunicorn

 🧠 What I learned from this

This project helped me understand:

* How frontend and backend connect
* How websites handle user input
* How to work with real-world data (not just examples)
* How to deploy a project online

😅 Challenges I faced
* Getting HTML templates to work properly
* Fixing errors when data wasn’t showing
* Understanding how deployment works
* Figuring out why things break (a lot 💀)

🌱 What I want to improve
* Make the summary smarter using real AI models
* Improve the UI (make it look cleaner)
* Add more features like file upload
